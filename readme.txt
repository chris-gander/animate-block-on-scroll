=== Animate Block On Scroll ===
Contributors:      Chris Gander
Tags:              block animate
Tested up to:      6.1
Stable tag:        1.0.0

A Gutenberg plugin that allows you to animate any block on scroll using AOS

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/animate-block-on-scroll` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
